They are for the community by the community and there are all the intentions of collaborating with upstream once underlying packages are into Guix proper. The idea is to make it as little effort as possible upstreaming code from this channel to Guix.\n</li><li> Services are configured with sane, secure, privacy aware defaults when possible.
